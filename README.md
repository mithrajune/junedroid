# junedroid
이 프로젝트는 Brut-Security의 BrutDroid 를 기반으로 수정 및 개선된 버전입니다. 본 소프트웨어는 원본과 동일하게 MIT 라이선스에 따라 배포됩니다. 사용, 수정, 배포가 자유롭지만 반드시 원저작권 및 라이선스 고지를 유지해야 합니다.
